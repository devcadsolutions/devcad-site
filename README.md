
# DevCAD Solutions - Static Site

This is a modern, high-performance static website for DevCAD Solutions, built with Next.js and optimized for GitHub Pages.

## Deployment to GitHub Pages

1. **Build the project**:
   ```bash
   npm run build
   ```
   This generates an `out` directory containing the static HTML/CSS/JS.

2. **Upload to GitHub**:
   Push the contents of the `out` folder to your GitHub repository (or use a GitHub Action to automate this).

3. **Entry Point**:
   The main entry point is `index.html`, located in the root of the exported directory.

## Features
- Fully responsive design
- Smooth entrance animations
- Static export (no server required)
- Optimized for fast loading
