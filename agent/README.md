# Agent Workspace

This folder stores project-specific guidance for future agent-assisted work on `devcad-site`.

## Structure

- `skills/`
  Project-local skills for recurring work such as UI polish, content editing, and GitHub Pages-safe implementation.
- `memory/`
  Durable notes about the site's architecture, design direction, deployment assumptions, and important decisions.

## Usage

- Add or update a `SKILL.md` file inside a skill folder when a workflow becomes repeatable.
- Add or update memory files when decisions are made that future work should respect.
- Keep entries short, specific, and current.

## Suggested conventions

- One skill per folder: `skills/<skill-name>/SKILL.md`
- One topic per memory file: `memory/<topic>.md`
- Prefer practical instructions over long narrative notes.
